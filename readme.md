
# 1. Install & Update

``` bash
$ npm install hexo -g
$ npm update hexo -g
$ hexo init
```

# 2. Command

``` bash
# new blog
$ hexo new "blog name"
$ hexo n "blog name"

# generate
$ hexo generate
$ hexo g

# server preview
$ hexo server -p port_id
$ hexo s

# deploy
$ hexo deploy --message "commit info"
$ hexo d
```
